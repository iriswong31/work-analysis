# 深根者计划对共富学堂的需求整理

> 来源：Iris 2026.2.3 提供

---

## 1. 概述

腾讯基金会与中国社会保障学会牵头发起了"深根者计划"，同时联合民政职业大学公益慈善研究院、清华大学公益慈善研究院、北京师范大学中国公益研究院、深圳国际公益学院（以下统称为联合机构）带动各地基金会面向社区工作人员及志愿者做公益培训。

**业务目标：** 将培训班管理、课程体系形成标准化、体系化，由线下培训逐步转为线上培训，以便扩大"深根者计划"的影响力。

### 系统用户

| 用户类型 | 说明 |
|---------|------|
| 系统管理用户 | 联合机构的数字化管理员 |
| 培训项目管理用户 | 各地基金会的培训管理员 |
| 学员用户 | 社区书记、社工、志愿者等 |

### 学员特点
- **来源**：各地政府推荐为主，志愿者主动报名为辅
- **培训结果预期**：非强制性学习，以任务引导及证书发放作为激励手段

### 当前业务情况
目前全部培训以线下为主，还未引入数字化工具，且课程内容较为零散，培训管理过程还未标准化，各合作机构及培训班管理过程参差不齐。

## 2. 需求梳理

### 2.1 原始需求记录

**需求收集信息：**
- 需求收集对象：王艳君、沈婧薇、Fancy Chan
- 需求收集时间：2026年1月14日~1月16日
- 需求收集方式：4次线下面对面沟通会议、2次线上交流会议

### 2.2 Coursera平台参考要点

在谈需求最初，艳君讲述了Coursera平台比较友好，其实用点：
1. 可以清晰看到每个课程的内容介绍、学习的计划、目标（可获得的证书）
2. 每个课程有纲要介绍，并可以选择章节快速学习，学习过程中有术语解释
3. 培训中为防止学员溜号，有不定时的课堂测验
4. 培训结束后有测试复习

### 2.3 深根者计划整体架构

四个联合发起方在整个培训过程中极为重要，从学员招募、培训课程管理、课程内容提供等层面均起到主导作用。

### 2.4 培训流程标准化雏形

通过与艳君团队的交谈过程中，发现其培训过程有标准化的雏形：

1. **联合机构分权管理**：联合机构下创立培训课程班，培训机构关注各培训项目的执行情况，培训课程班负责具体培训执行

2. **培训课程标准化**：
   - 学员签到（字段标准化：学员姓名、单位、签到）
   - 课程纲要发放
   - 听课
   - 作业/测验（暂时还未执行过）

3. **三阶段培训流程**：
   - 基础课程学习
   - 线下走访/研学
   - 考核并颁发资质（还未执行过）

4. **共性培训课程内容**：
   - 《深根者计划介绍》
   - 《社区慈善是什么及发展趋势》
   - 《关爱平台介绍》

## 3. 关于"一起学"平台的了解

腾讯基金会先后做了两个版本的"一起学"平台：

### 乐享版"一起学"
- 最开始使用乐享搭建了学习平台
- 由于乐享平台在个性化培训过程无法满足需求，并且无法生成独立的小程序
- 主要用于：线上学习
- 小程序主体：深圳市腾讯计算机系统有限公司
- 管理后台：https://yiqixue.lexiangla.com/

### 自研版"一起学"
- 后自研开发了新版一起学平台
- 主要用于：机构管理、线下培训活动管理
- 小程序主体：腾讯公益慈善基金会
- 管理后台：https://yiqixue-admin.tencentfoundation.org/management/home

**并行使用原因**：由于两个平台各有优势，所以在一段时间中两个平台并行使用。

## 4. 需求满足情况结论

### 4.1 三平台对比

| 核心需求 | 共富学堂 | 乐享版"一起学" | 自研版"一起学" |
|---------|---------|---------------|---------------|
| 各培训项目需要独立品牌（独立小程序） | ✗ | ✗ | ✓ |
| 支持有层级的合作机构模式 | ✓ | ✗ | ✗ |
| 支持培训项目过程管理 | ✓ | ✓ | ✗ |
| 课程管理（线上视频课、直播课） | ✓ | ✓ | ✗ |
| 线下课程管理（报名、签到） | ✓ | ✓ | ✗ |

### 4.2 各平台评价

**共富学堂**
- 具备机构管理、培训项目管理、学习任务管理、线上课程学习等功能，基本满足深根者计划的当前需求
- 后续还需要在具体项目中试用再看是否需要优化

**乐享版"一起学"**
- 产品结构大而全，产品较为复杂
- 缺少对机构层次化的管理，对培训管理过程的支持缺少特有需求的支持
- 无法满足深根者计划需求
- 其中的部分课程内容，可以提炼出来供深根者计划复用

**自研版"一起学"**
- 产品较为单一，当前更偏重机构管理、活动报名、线上课程观看功能
- 缺少有层次和分权的机构管理模式，也没有培训过程管理及培训签到等功能
- 无法满足深根者计划需求
- 其中的部分课程内容，可以提炼出来供深根者计划复用

### 4.3 推荐结论

对比共富学堂、乐享版"一起学"、基金会自研版"一起学"，**共富学堂更符合"深根者计划"的需求**，后续以共富学堂为线上平台优化，其他两个产品中的课程挑拣出放入共富学堂中。

**关于Coursera**：由于其只面向C端用户，无机构管理以及培训项目的概念，所以无法复用，但其培训课程学习能力可以做为后续产品迭代参考。
