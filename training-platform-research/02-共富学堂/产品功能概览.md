# 共富乡村学堂 - 产品功能概览

> 资料来源：共富乡村学堂产品介绍-1229.pdf
> 整理时间：2026年2月3日

## 一、产品定位

**面向乡村经营人才的线上学习与交流平台——学而有效，经营有成**

## 二、产品架构（C端小程序）

共富学堂小程序包含以下核心模块：

| 模块 | 功能说明 |
|-----|---------|
| **官方首页** | 学堂新闻和动态、学员案例、培养体系、推荐课程 |
| **机构管理** | 培育项目数据展示（培育兴农典型/创新/火种/导师/实训营数量） |
| **培训项目管理** | 计划站点页、学习阶段展示（基础培训→线上拓展→线下实训→线下互访→视频号投递→太雷理） |
| **线上学习** | 课程学习、导师介绍、课程目录 |
| **互动交流** | 微信小店经营专区、攻略分享、评论互动 |

## 三、培训管理能力（SaaS模式）

### 3.1 计划站点服务模块

为每个合作方提供独立的计划站点页，包含：
- 培养方案学习
- 人物案例学习
- 计划公告通知
- 新闻资讯动态
- 活动线上报名
- 热门课程学习

### 3.2 培训全流程管理

**从报名甄选至结业颁证的线上培训管理流程：**

| 环节 | 功能 |
|-----|------|
| **学员报名** | 在线填写报名表、信息收集 |
| **学习计划** | 查看学习计划安排、阶段任务解锁、进度展示 |
| **线上学习** | 课程介绍、课程目录、导师介绍、视频学习 |
| **线上测验** | 完成度显示、限时考试 |
| **颁发证书** | 电子证书生成、可分享 |

### 3.3 仪式感设计

- **录取通知** - 弹窗通知+查看详情
- **分享通知书** - 可保存图片/分享好友
- **结业证书** - 带编号的正式证书，可分享

### 3.4 任务与进度管理

- 待学任务展示
- 学习任务列表（多阶段：基础理论→线上拓展→线下实训→线下互访→视频号投递→太雷理）
- 完成进度可视化（百分比+任务解锁）

## 四、社区交流功能

**打造学员间的实名交流互动场域：**

| 功能 | 说明 |
|-----|------|
| **经验交流** | 广场发帖、学习心得分享、评论互动 |
| **人脉咨询** | 学员名录、职位/地区信息、可联系 |
| **需求合作** | 发布合作需求、互益付费合作 |

## 五、签到功能

**无纸化报到、签到场景：**

- **入学报到** - 地理位置签到（已进入范围内）
- **学习签到** - 村庄实训签到、位置校验
- **签到码** - 二维码扫码签到

## 六、课程形式

| 类型 | 说明 |
|-----|------|
| **点播课程** | 视频课、课程介绍、导师介绍、立即观看 |
| **直播课程** | 预约直播、倒计时、直播签到 |
| **线下课程** | 签到地址、线下报名 |

## 七、作业与互动

- **作业任务** - 开班后按时间节点发布作业
- **发布入口** - 学员可发布动态、选择话题、上传图片/视频
- **心得分享** - 驻村心得、学习感悟展示

## 八、活动管理

| 类型 | 功能 |
|-----|------|
| **线下活动报名** | 活动介绍、主讲人介绍、活动流程、立即报名 |
| **线上直播活动** | 预约线上直播、倒计时 |
| **活动报名表单** | 自定义字段（姓名、手机、出行方式、住宿、单选、多选等） |

## 九、考核评估

| 类型 | 说明 |
|-----|------|
| **课程测验** | 选择题，学习后即时测验 |
| **问卷反馈** | 满意度调查、培训效果评估 |
| **阶段考试** | 限时考试、多选题支持 |

## 十、培训前中后流程

```
培训前
├── 官方站点页
├── 站点模块名称
├── 项目学习页
├── 学员入驻-内部选定
└── 学员入驻-招募报名

培训中
├── 点播/直播课程学习
├── 活动
├── 考试
├── 签到
└── 问卷

培训后
├── 电子证书
└── 学员身份
```
