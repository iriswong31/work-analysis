# 耕耘者学习平台 - 产品概览

> 来源：Iris 2026.2.3 提供
> 
> 品牌定位：为村耕耘者知识分享平台（乡村版百科全书）
> 
> 主体：腾讯SSV / 为村耕耘者

---

## 一、平台架构（三端）

耕耘者学习平台由三个端组成：

```
┌─────────────────────────────────────────────────────────────────┐
│                     耕耘者学习平台架构                           │
├─────────────────┬─────────────────┬─────────────────────────────┤
│    运营后台      │   培训机构后台   │         C端学习平台          │
│  （平台运营）    │  （培训机构）    │         （学员）             │
├─────────────────┼─────────────────┼─────────────────────────────┤
│ 内容/课程发布    │ 班级管理        │ 课程内容                     │
│ 权限管理        │ 课程管理        │ 班级页签到考核等              │
│                 │ 机构申请        │                              │
└─────────────────┴─────────────────┴─────────────────────────────┘
```

---

## 二、运营后台

### 基本信息
- **使用对象**：公司内运营和中央农广校内容运营人员
- **产品形态**：PC端后台

### 主要功能
- 内容发布
- 内容审核
- 用户反馈管理
- 权限管理

### 培训内容配置
可配置的内容类型：
- 课程
- 考试
- 作业
- 班级页

### 内容结构体系

```
课节 ➡️ 主题 ➡️ 专题 ➡️ 学习计划（含考试） ➡️ 班级页
```

| 层级 | 说明 |
|------|------|
| **课节** | 最基础的学习单位，可能为视频或图文课程 |
| **主题** | 由多个课节组成，形成具体的学习模块 |
| **专题** | 多个主题的集合，形成一个综合性课程 |
| **考试** | 线上考试题库/答案配置和管理，题型主要是选择题 |
| **学习计划** | 计划中可以包括课程、考试等配置 |
| **班级页** | 为每个班级设定的管理页面 |

### 班级页功能
- 查询培训信息
- 线上课程
- 学习计划
- 作业
- 签到
- **打通了村干部在善治美的数据**

### 班级页使用流程
1. 运营后台配置班级页模板
2. 培训机构班主任在培训后台选择对应模板
3. 班主任可查看学员的各项进度

---

## 三、培训机构后台

### 基本信息
- **使用对象**：公司培训项目组与培训机构的班主任
- **产品形态**：PC端后台（推测）

### 主要功能

| 功能模块 | 说明 |
|---------|------|
| **数据看板** | 展示线上线下培训进度数据，线上课程/作业完成率、线下签到率等 |
| **班级管理** | 管理和创建培训班级，设置班级的名称、培训内容、报名情况、学员进度等 |
| **课程管理** | 主要是提供给培训机构线下培训时可以打印的资料包 |
| **合作伙伴** | 各培训机构名录 |
| **审批** | 审核各机构开班申请，通过后，班主任才能建班 |

### 审批流程
```
培训机构提交开班申请 → 审批通过 → 班主任可以建班
```

## 四、C端学习平台

### 基本信息
- **使用对象**：学员（村干部等）
- **产品形态**：微信小程序（"为村耕耘者"）
- **定位**：乡村版百科全书

### 核心功能模块

| 功能 | 说明 |
|------|------|
| **学政策** | 传递国家最新三农政策，并提供权威专家解读 |
| **耘课堂** | 覆盖"三农"政策、农业技术和通识能力等视频课程 |
| **写日记** | 实时记录学习心得和治理经验 |
| **乡村振兴100问** | 从0-1拆解标杆村庄振兴关键诀窍 |
| **线下培训** | 了解培训总览，全国开班情况，一键报名参加培训 |

### 页面结构（从截图分析）
- 首页：学政策、耘课堂、推荐、乡村振兴100问等Tab
- 课程详情页
- 问答/回答详情页
- 日记详情页
- 班级页（签到、考核等）

### 内容特色
- 乡村治理骨干、新时代农业经营主体相关内容
- 村支书们都在学的好课推荐
- 乡村治理系列内容
- 猜你想学（个性化推荐）

---

## 五、平台特点总结

| 特点 | 说明 |
|------|------|
| **三端分离** | 运营后台、培训机构后台、C端学习平台职责清晰 |
| **多层级内容结构** | 课节→主题→专题→学习计划→班级页，支持灵活组合 |
| **线上线下结合** | 支持线上课程学习 + 线下培训签到 |
| **机构分权管理** | 培训机构可独立管理自己的班级 |
| **审批流程** | 开班需要审批，确保培训质量 |
| **数据打通** | 与善治美数据打通（村干部数据） |
| **内容丰富** | 政策解读、视频课程、问答、日记等多种形式 |
