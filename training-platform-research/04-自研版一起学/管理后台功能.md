## 二、课程管理功能

### 2.1 课程列表
- 显示：排序、序号、课程封面、课程名称、课程时间、任务数、操作
- 操作：编辑、删除、新增课程

### 2.2 课程编辑字段
| 字段 | 说明 |
|-----|------|
| 封面图 | 上传课程封面 |
| 课程名称 | 文本输入 |
| 课程时间 | 起止时间范围 |
| 任务数 | 数字（如1000） |
| 参与人数 | 数字（如30000） |
| 课程内容 | 富文本编辑器（支持H1-H3、加粗、斜体、下划线、对齐等） |
| 课程链接 | 跳转到乐享版课程（如 https://yiqixue.lexiangla.com/） |

**注意：课程内容实际是跳转到乐享版，自研版本身不承载课程视频**

## 三、活动管理功能

### 3.1 活动列表看板
- 全部活动数
- 待审核数
- 未开始数
- 进行中数
- 已结束数

### 3.2 活动筛选条件
- 主题
- 性质
- 类型
- 活动时间
- 状态
- 主办机构
- 提交时间
- 推荐到首页Banner

### 3.3 四种活动类型
| 类型 | 适用场景 |
|-----|---------|
| **创建单场活动** | 研讨会议、交流会、专题沙龙、研学活动、工作坊、公益市集、体验活动、公益路演等 |
| **创建系列活动** | 全年不定期举办的多场活动，活动场次和类型不固定 |
| **创建大型会议** | 集中时间举办多场活动，包括开幕主论坛、闭幕主论坛、若干平行会议等 |
| **创建峰会** | 集中时间举办大型峰会 |

### 3.4 活动创建7步流程
1. **选择创建的会议类型** - 单场/系列/大型会议/峰会
2. **创建基础信息，保存草稿** - 活动类型、模版、基础信息、机构信息、精选照片
3. **管理嘉宾信息** - 邀请嘉宾出席或添加嘉宾信息
4. **创建环节或议程信息** - 新增环节/新增会议活动
5. **上传常见问题和会议手册** - FAQ配置
6. **提交信息并上架** - 提交上架后官网显示
7. **上传成果信息** - 直播录屏、资讯报道、相关报告等

### 3.5 活动详情管理Tab
| Tab | 功能 |
|-----|------|
| 基本信息 | 活动主题、类型、模版、时间、地点、举办形式 |
| 议程管理 | 活动议程配置 |
| 海报与邀请函管理 | 活动宣传物料 |
| 嘉宾管理 | 嘉宾信息维护 |
| 参会管理 | 参会人员管理 |
| 常见问题 | FAQ配置 |

## 六、平台定位分析

**自研版"一起学"的核心定位：**
1. **机构管理中心** - 管理合作公益机构
2. **活动/会议管理** - 大型公益活动/会议的全流程管理（偏"办会工具"）
3. **资源链接** - 专家库、机构库、案例库、工具包
4. **资助计划展示** - 项目信息展示+外链报名

**局限性：**
- 课程内容依赖乐享版，自身不具备课程管理能力
- 报名功能依赖外链（微信公众号），非平台内闭环
- 缺少培训班/学习任务管理能力
- 缺少签到打卡、学习进度追踪等培训过程管理

**适合场景：**
- 公益行业会议/论坛组织
- 公益机构资源链接
- 项目信息展示与招募引流

**不适合场景：**
- 系统化培训班管理
- 学员学习过程追踪
- 线上线下结合的培训项目
