# 网文创作工坊 📝

**女频年代文 · 50万字企划**

一个4-Agent协作的网文创作平台，部署在 GitHub Pages。

## 四个 Agent

| 编号 | 名称 | 角色 | 职责 |
|------|------|------|------|
| 1 | 🔍 扫榜官 | 数据采集员 | 爬取番茄小说排行榜Top10，总结爽点 |
| 2 | ✍️ 笔刀手 | 小说作者 | 基于数据创作50万字大纲，每日更新4000字 |
| 3 | 📝 毒舌编辑 | 网文编辑 | 专业打分审核，打回不合格内容 |
| 4 | 💖 爽感鉴定官 | 女频读者 | 评估爽感是否足够，不够则打回 |

## 协作流程

```
扫榜官 → 笔刀手 → 毒舌编辑 → 爽感鉴定官
                      ↑  打回  ↓
                      ←--------←
```

## 使用方式

所有内容生成由 WorkBuddy AI 在后台完成。用户只需在 WorkBuddy 对话中说：
- `@笔刀手 开始创作` — 启动大纲创作
- `@毒舌编辑 审核` — 触发编辑审核
- `@爽感鉴定官 评测` — 触发爽感评测

## 技术栈

- 纯 HTML + CSS + JavaScript（零构建依赖）
- JSON 数据驱动
- GitHub Pages 部署
- WorkBuddy AI 内容生成
