# 深根者培训

> 负责人最关注的核心项目，社区慈善人才培育的旗舰计划

---

## 一、项目概述

### 1.1 基本信息

| 维度 | 内容 |
|------|------|
| **项目全称** | 深根者计划——社区慈善人才培养项目 |
| **启动时间** | 2024年9月4日（第十个中华慈善日） |
| **发起方** | 中国社会保障学会 + 腾讯公益慈善基金会 |
| **响应政策** | 民政部等七部委"慈善文化"进社区、进家庭倡导 |

### 1.2 联合发起机构

```

│  学术支撑                                                        │
│  ├── 清华大学公益慈善研究院                                      │

│  ├── 深圳国际公益学院                                            │
```

### 1.3 培养目标

**首期规模**：350名社区慈善人才

**覆盖城市**：京津冀、深圳、杭州、长沙、重庆、酒泉（共8个城市和地区）

**核心能力**：
- **社区行动力** - 能够在社区落地项目
- **资源动员力** - 能够整合各方资源
- **持续成长力** - 能够持续学习和迭代

---

## 二、培养对象

### 2.1 多元角色覆盖

深根者计划打破传统培训的单一对象限制，覆盖社区内多元潜力角色：

| 角色类型 | 说明 |
|---------|------|
| 县（区）民政主管领导 | 政策制定和资源调配者 |
| 街道社区干部 | 基层治理执行者 |
| 社工 | 专业服务提供者 |
| 居民志愿者达人 | 社区活跃分子 |
| 社区社会组织主理人 | 公益组织负责人 |
| 青年创业者 | 创新力量 |
| 社区小店经营者 | 商业+公益融合者 |

### 2.2 组队要求

- 鼓励**跨身份组队**
- 深圳要求**3-5人同社区组队**
- 强调**多元角色协同**

---

## 三、培训模式

### 3.1 三阶段培养体系

```
┌─────────────────────────────────────────────────────────────────┐
│                    三阶段培养体系                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  第一阶段：理论筑基                                              │
│  ├── 系统培训（3天集中授课）                                    │
│  ├── 《慈善法》培训                                              │
│  ├── 社区慈善运营方法论                                          │
│  └── 专家授课 + 案例分析                                         │
│                                                                  │
│  第二阶段：实践强基                                              │
│  ├── 跨区域参访（如赴广东顺德）                                  │
│  ├── 实地考察优秀社区                                            │
│  ├── 沉浸式体验                                                  │
│  └── 实战演练                                                    │
│                                                                  │
│  第三阶段：成效扩基                                              │
│  ├── 微项目落地支持                                              │
│  ├── 路演竞争种子基金（最高1.2万元）                            │
│  ├── 纳入人才库                                                  │
│  └── 形成可持续实践网络                                          │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 核心理念

> **"立足社区、动员社区、服务社区、提升社区"**

### 3.3 培训特色

| 特色 | 说明 |
|------|------|
| **实战导向** | 强调"学完必须带来社区改变" |
| **数字化赋能** | 整合腾讯数字关爱平台等工具 |
| **跨域交流** | 组织跨区域参访学习 |
| **项目落地** | 微项目+种子基金支持 |

## 四、落地进展

### 4.1 各地开班情况

| 城市 | 时间 | 合作方 | 规模 |
|------|------|--------|------|
| **北京** | 2024年9月4日 | 启动仪式 | - |
| **重庆** | 2024年11月26日 | 清华大学公益慈善研究院 + 重庆市慈善联合总会 | 50名学员 |
| **长沙** | 2024年12月23日 | 长沙市民政局 | 赋能实践基地启动 |
| **杭州** | 2025年11月2日 | 杭州赋能实践基地 | 首期培训班开班 |

### 4.2 重庆班详情

**发起方**：清华大学公益慈善研究院 + 重庆市慈善联合总会

**学员构成**：
- 社会组织代表
- 社区干部
- 基层社工
- 共50名学员

**培训规划**：
1. 第一期：理论筑基（2024年11月）
2. 第二期：实践强基 - 赴广东顺德参访（2025年12月下旬）
3. 第三期：成效扩基 - 微项目落地

### 4.3 长沙班详情

**特色**：
- 参访观摩 + 集中授课
- 为期3天
- 专业理论知识 + 科学实践指导

**目标**：
> "成为连接政府、慈善组织、爱心企业与社区居民之间的关键一环，探索更加可持续的服务模式，打造一批'小而美'的社区慈善品牌项目"

## 五、生态伙伴

### 5.1 美团公益基金会

美团公益基金会参与深根者计划，带来**袋鼠宝贝之家**项目：

| 指标 | 数据 |
|------|------|
| 覆盖城市 | 19个（北京、上海、深圳、重庆等） |
| 建立空间 | 35个（运营中30个） |
| 服务人次 | 22.2万人次 |
| 服务对象 | 新就业群体家庭的未成年子女 |

**项目定位**：在城市社区建设儿童友好空间，为新就业群体家庭提供社区教育服务

## 六、社会意义

### 6.1 激活基层治理

- 培育"毛细血管"型人才
- 长沙180名学员带动1356支社区基金
- 缩小政府服务与居民需求的"最后一公里"差距

### 6.2 构建慈善生态

- 打破资源孤岛
- 广州"公益集市"链接企业资源
- 形成"居民互助-商户参与-社会组织协同"的可持续网络

### 6.3 文化传承创新

- 挖掘本土文化
- 杭州金鱼井社区"善良公社"
- 将传统邻里互助升级为数字化公益

## 七、解决痛点

| 痛点 | 解决方案 | 案例 |
|------|---------|------|
| **人才断层** | 专业培训路径 | 深圳非专业人士发起孤独症融合项目 |
| **服务低效** | 数字化工具 | 腾讯小程序实现47万件"村民说事"线上化 |
| **可持续性差** | 种子基金+跨域合作 | 长沙与广深经验互通 |

## 八、成功案例

### 案例一：深圳蛇口社区

- 将小店改造为"善意空间"
- 年服务超200场
- 商业+公益融合典范

### 案例二：长沙开福区东华社区

- 通过积分制动员80%居民参与环保
- 居民自治典范

### 案例三：杭州金鱼井社区

- "善良公社"项目
- 挖掘本土文化
- 传统邻里互助升级为数字化公益
