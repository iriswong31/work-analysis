## 一、战略演进

### 1.1 腾讯公益发展历程

| 时间 | 里程碑 | 意义 |
|------|--------|------|
| 2002年 | 为广东清远山区小学捐赠电脑 | 公益起点 |
| 2007年 | 腾讯公益慈善基金会成立 | 组织化运作 |
| 2010年 | 成立企业社会责任部 | 战略化布局 |
| 2015年 | 第一届"99公益日"启动 | 全民公益创新 |
| 2019年 | "用户为本，科技向善"使命愿景 | 价值观升级 |
| **2021年** | **成立SSV事业部，首期投入500亿** | **战略级升级** |

### 1.2 SSV战略定位

2021年4月，腾讯成立**可持续社会价值事业部**（Sustainable Social Value Organization，简称SSV），将"科技向善"从使命愿景落地为具体执行部门。

**核心理念**：
> "超越短期的盈利目标，从基于商业原则的市场竞争转向基于社会原则的可持续价值创造，在企业实践中体现社会原则优先。"

**投入规模**：首期500亿元人民币

## 二、公益平台能力

### 2.1 腾讯公益平台

作为民政部首批指定的互联网募捐信息平台之一，腾讯公益平台已成为**全球最大的互联网募款平台之一**。

**核心数据（截至2024年12月）**：

| 指标 | 数据 |
|------|------|
| 历史善款总额 | 337亿元+ |
| 历史爱心人次 | 13亿人次+ |
| 入驻公益机构 | 3.3万+ |
| 帮助项目数 | 13.2万+ |

### 2.2 数字关爱平台

2024年9月正式上线，基于微信支付的数字化资助工具。

**核心功能**：
- **关爱券/金**：将慈善资金转化为公益用途的微信支付消费券/金
- **精准发放**：资助方可精准发放给受助人
- **场景覆盖**：救灾资金发放、老人社区就餐、孤独症儿童社区融合等

**定位**：
> "帮对人，用对钱，建通路"的数字化公益服务平台

### 2.3 数字备灾救灾体系

2025年5月发布"2025数字备灾救灾支持体系"：

**共建机制**：
- 腾讯公益慈善基金会
- 中国红十字基金会
- 壹基金
- 爱德基金会
- 中华社会救助基金会
- 云南省慈善总会

**覆盖范围**：计划在全国16省市推进数字备灾救灾合作

## 三、人才培养体系

### 3.1 深根者计划（重点）

**发起方**：中国社会保障学会 + 腾讯公益慈善基金会

**联合发起**：
- 民政职业大学
- 清华大学公益慈善研究院
- 北京师范大学中国公益研究院
- 深圳国际公益学院

**启动时间**：2024年9月4日（第十个中华慈善日）

**培养目标**：
- 首期培养350名社区慈善人才
- 覆盖京津冀、深圳、杭州、长沙、重庆、酒泉共8个城市和地区
- 培育具备**社区行动力、资源动员力、持续成长力**的复合型人才

**培养对象**：
- 县（区）民政主管领导
- 街道社区干部
- 社工
- 居民志愿者达人
- 社区社会组织主理人
- 青年创业者
- 社区小店经营者

**核心理念**：
> "立足社区、动员社区、服务社区、提升社区"

**培训模式**：
1. **理论筑基** - 系统培训
2. **实践强基** - 跨区域参访（如赴广东顺德）
3. **成效扩基** - 微项目落地支持

**落地进展**：
- 2024年12月：长沙赋能实践基地启动
- 2024年11月：重庆班开班

### 3.2 数字人才开放计划

**主办方**：腾讯公益 + 中国西部人才开发基金会

**培训内容**：
- 慈善组织法律风险防范
- 舆情管理与危机公关
- 互联网筹款技能提升
- 慈善组织财务合规管理

**覆盖范围**：陕西、山东、重庆等多省市

### 3.3 数字强基资助计划

**目标**：推动区域内数字化公益高质量发展

**资助方向**：
- 公益项目规范化管理培训
- 筹款动员能力提升
- 透明合规能力建设

## 四、乡村振兴布局

### 4.1 为村共富乡村项目

**发起方**：腾讯SSV为村发展实验室

**核心模式**："机制+人才+数字化"的内生型共富乡村系统化解决方案

**原则**：政府主导、农民主体、腾讯助力、社会共创

**核心抓手**：乡村职业经理人（乡村CEO）培养

**成果（截至2024年底）**：
- 推广至全国17个省份309个县
- 乡村CEO成为人力资源和社会保障部发布的新职业之一
- 经验走向非洲等国际市场

**典型案例**：
- 江门恩平昌梅村：乡村CEO策划"旅游+研学"项目，为村集体带来近35万元收入

## 六、战略意图分析

### 6.1 为什么是社区慈善？

1. **响应国家战略** - 2024年新《慈善法》明确鼓励社区慈善
2. **基础设施定位** - 微信支付+小程序天然适合社区场景
3. **用户连接** - 社区是连接C端用户的最后一公里
4. **生态构建** - 政府、社区、居民、商户的多方协同

### 6.2 深根者计划的战略价值

深根者计划是腾讯在社区慈善领域的**人才基础设施**：

| 维度 | 价值 |
|------|------|
| 政策响应 | 响应民政部"慈善文化进社区、进家庭"倡导 |
| 人才培育 | 培养懂政策、通理论、会实操的复合型人才 |
| 生态构建 | 形成可持续的实践网络与在地经验 |
| 品牌影响 | 腾讯公益在基层治理领域的品牌渗透 |

### 6.3 对你的启示

作为腾讯内部员工，社区慈善领域的机会点：

1. **技术赋能** - 数字关爱平台、小程序等工具的优化
2. **培训支持** - 深根者计划的课程设计、数字化工具教学
3. **产品创新** - 社区慈善基金管理、志愿者匹配等场景
4. **生态连接** - 连接政府、社区、公益组织的枢纽角色
