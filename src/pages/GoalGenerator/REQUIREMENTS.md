# 2026 年度目标生成器 - 需求文档

## 项目概述
一个可视化的年度目标卡片生成器，用户输入年度目标等信息后，生成多种风格的精美卡片供下载分享。

---

## 核心需求

### 1. 金句系统
- **不重复规则**：每次生成时，5张卡片（FullGoalCard、TemplateA/B/C/D）上的金句必须各不相同
- **随机性**：每次刷新页面，金句组合应随机变化
- **数据来源**：金句来自 `data/quotes.ts`
- **实现方式**：PreviewPage 使用 `getRandomQuotes(5)` 生成5个不重复金句，通过 `quote` prop 分别传递给各模板

### 2. 数据展示
- **动态适配**：用户输入多少条目标就展示多少条，不能限制条数
- **场景卡D**：显示用户输入的所有年度目标，不设上限
- **空值处理**：用户没有输入时，不显示默认占位内容（避免误导）

### 3. 数据解析（智能 parseList）
- **统一工具函数**：`utils/parseList.ts` 提供智能解析
- **支持的分隔方式**：
  - 换行符（`\n`、`\r\n`、`\r`）
  - 中文标点（顿号`、`、中文分号`；`、中文逗号`，`）
  - 英文分号（`;`）
  - 数字序号（`1. xxx 2. xxx` 或 `1、xxx 2、xxx`）
- **容错机制**：如果换行分割只得到1条，自动尝试其他分隔符

### 4. 移动端适配
- **响应式设计**：页面在手机端可正常使用
- **保存到相册**：手机端点击下载时，弹出预览图片，提示用户长按保存
  - 自动检测移动设备
  - 显示预览弹窗，用户长按图片即可保存到相册

## 技术实现要点

### 卡片模板结构
- `FullGoalCard`：完整目标卡片（第5个金句）
- `TemplateA`：四象限布局（第1个金句）
- `TemplateB`：晨间提醒卡（第2个金句）
- `TemplateC`：时间轴 + 月度专注（第3个金句）
- `TemplateD`：年度宣言海报（第4个金句）

### 数据存储
- 使用 `localStorage` 持久化用户输入
- key: `goal-generator-2026`
- 重置功能需清除 localStorage

### 文件结构
```
src/pages/GoalGenerator/
├── utils/
│   └── parseList.ts          # 统一的列表解析工具
├── data/
│   ├── quotes.ts             # 金句库
│   └── inspirations.ts       # 灵感提示
├── components/
│   └── cards/
│       ├── TemplateA.tsx     # 四象限模板
│       ├── TemplateB.tsx     # 晨间提醒模板
│       ├── TemplateC.tsx     # 时间轴模板
│       ├── TemplateD.tsx     # 年度宣言模板
│       └── FullGoalCard.tsx  # 完整卡片
├── hooks/
│   └── useGoalForm.ts        # 表单状态管理
└── PreviewPage.tsx           # 预览页面（含下载功能）
```
