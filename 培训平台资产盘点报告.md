## 📋 报告概述

本报告基于腾讯基金会"深根者计划"的实际需求，对现有4个培训平台进行全面盘点分析。采用4模块分析框架（课程管理、班务管理、角色管理、数据看板），为后续平台选择和功能优化提供决策支持。

### 关键数据

| 指标 | 数量 |
|------|------|
| 培训平台 | 4个 |
| 核心模块 | 4个 |
| 课程资源 | 1580+ |
| 专业课程 | 13门 |

### 分析目标

通过系统性盘点，识别各平台在课程管理、班务管理、角色管理、数据看板四个维度的能力差异，为"深根者计划"选择最适合的技术方案，并明确内容资源的复用策略。

## 🏢 平台基本信息

### 1. 共富学堂

**定位：** 腾讯SSV为村共富乡村旗下，面向乡村经营人才的线上学习与交流平台

**核心特色：**
- SaaS化培训管理
- 多层级机构管理
- 完整培训流程
- 线上线下结合

### 2. 一起学（乐享版）

**定位：** 腾讯基金会基于乐享搭建的培训版本，主要用于线上学习

**核心特色：**
- 13门专业课程
- 148节课程内容
- 5576.5分钟内容
- 乐享平台承载

### 3. 一起学（自研版）

**定位：** 腾讯基金会自研开发的培训平台，主要用于机构管理、线下培训活动管理

**核心特色：**
- 机构管理中心
- 活动会议管理
- 资源链接平台
- 项目展示招募

### 4. 耕耘者学习平台

**定位：** 农村治理培训平台，定位为乡村版百科全书

**核心特色：**
- 1568门课程
- 三端分离架构
- 多层级内容结构
- 与善治美数据打通

## 🎯 深根者需求匹配分析

### 深根者计划核心需求

**业务目标：** 腾讯基金会与中国社会保障学会等联合机构发起"深根者计划"，面向社区工作人员及志愿者进行公益培训，目标是将培训班管理、课程体系标准化、体系化，由线下培训逐步转为线上培训。

**关键特征：**
- **三类用户：** 系统管理员、培训管理员、学员
- **三阶段流程：** 基础学习→线下走访→考核颁证
- **多层级管理：** 联合机构→地方基金会→培训班
- **标准化培训：** 签到、纲要、听课、作业、考核

### 需求匹配度分析

| 核心需求 | 共富学堂 | 一起学（乐享版） | 一起学（自研版） | 耕耘者学习平台 |
|----------|----------|------------------|------------------|----------------|
| 多层级机构管理 | ✓ | ✗ | ✗ | ✓ |
| 培训项目管理 | ✓ | ◐ | ✗ | ✓ |
| 线上课程学习 | ✓ | ✓ | ✗ | ✓ |
| 线下活动管理 | ✓ | ✗ | ✓ | ✓ |
| 签到打卡功能 | ✓ | ✗ | ✓ | ✓ |
| 考核认证体系 | ✓ | ◐ | ✗ | ✓ |
| 独立品牌小程序 | ✗ | ✗ | ✓ | ✗ |

### 匹配度评估结论

**最佳匹配：共富学堂** - 在多层级机构管理、培训项目管理、线上线下结合等核心需求上表现最佳，基本满足深根者计划的当前需求。

**次优选择：耕耘者学习平台** - 功能完整度高，但可能存在定制化程度不够的问题。

**内容补充：一起学（乐享版）** - 虽然平台能力有限，但其专业课程内容可作为重要的内容资源补充。

## 📋 结论与建议

### 🎯 平台选择建议

**推荐方案：以共富学堂为主平台**

**理由：**
- 共富学堂在4个核心模块中表现最均衡，特别是在多层级机构管理和培训项目管理方面优势明显
- 已具备深根者计划所需的大部分核心功能，可快速投入使用
- 支持线上线下结合的培训模式，符合深根者计划的业务流程

### 📚 内容资源整合建议

**多平台内容资源整合策略：**

1. **核心课程迁移：** 将一起学（乐享版）的13门专业课程迁移至共富学堂，作为核心课程体系
2. **选择性整合：** 从耕耘者学习平台筛选与社区慈善相关的优质课程内容
3. **定制开发：** 开发深根者计划专属课程：计划介绍、社区慈善发展趋势、关爱平台介绍等

### 🔧 功能优化建议

**共富学堂需要完善的功能模块：**

1. **数据看板模块：** 增强数据可视化能力，完善报表导出功能
2. **考核认证：** 完善在线考试系统，优化证书颁发流程
3. **互动社区：** 加强学员间的交流互动功能
4. **移动端优化：** 提升小程序用户体验

### 📈 实施路径建议

**分阶段实施策略：**

**第一阶段（1-2月）：** 共富学堂基础功能测试，小规模试点
- 与艳君团队合作，进行平台功能验证
- 完成基础配置和权限设置
- 小范围学员试用，收集反馈

**第二阶段（3月）：** 内容资源迁移整合，社区关爱节全流程试用
- 完成一起学课程内容迁移
- 开发深根者专属课程
- 社区关爱节活动全流程测试

**第三阶段（4月后）：** 基于试用反馈优化功能，正式推广使用
- 根据试用反馈优化平台功能
- 完善数据看板和报表功能
- 正式推广到更多地区和机构
